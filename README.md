# Student Management System
A simple and interactive **Student Management System** built using **HTML, CSS, and JavaScript**. This project allows users to add and display student details through a user-friendly web interface.

## Features
- Add student details:
  - Student Name
  - Roll Number
  - Department
- Display added student records dynamically
- Simple and responsive user interface
- Interactive functionality using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
student-management-system/
│── index.html
│── style.css
│── script.js
│── README.md

## How to Run
1. Download or clone this repository
2. Open `index.html` in your browser
3. Enter student details
4. Click **Add Student** to display the record

## Project Purpose
This project was created to practice frontend development concepts including:
- Form handling
- DOM manipulation
- Event handling
- Basic UI design
